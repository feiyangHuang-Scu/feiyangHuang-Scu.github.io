时间复杂度：O(N)O(N)，其中 NN 是字符串的长度。左指针和右指针分别会遍历整个字符串一次。

空间复杂度：O(|\Sigma|)O(∣Σ∣)，其中 \SigmaΣ 表示字符集（即字符串中可以出现的字符），|\Sigma|∣Σ∣ 表示字符集的大小。在本题中没有明确说明字符集，因此可以默认为所有 ASCII 码在 [0, 128)[0,128) 内的字符，即 |\Sigma| = 128∣Σ∣=128。我们需要用到哈希集合来存储出现过的字符，而字符最多有 |\Sigma|∣Σ∣ 个，因此空间复杂度为 O(|\Sigma|)O(∣Σ∣)。

3.1-Hashset

定义方法

HashSet的一些常用方法：

添加元素：

hashset.add(E e)：返回boolean型，如果此 set 中尚未包含指定元素，则添加指定元素；如果此 set 已包含该元素，则该调用不更改 set 并返回 false。

删除元素：

hashset.clear()：从此 set 中移除所有元素。

hashset.remove(Object o)：如果指定元素存在于此 set 中，则将其移除。

hashset.isEmpty()：如果此 set 不包含任何元素，则返回 true。

hashset.contains(Object o)：如果此 set 包含指定元素，则返回 true。

hashset.size()：返回此 set 中的元素的数量（set 的容量）。

54-螺旋矩阵

该题从一拿到开始，就模拟着走，最终设置了四个变量left, right, top, bottom，分别记录上下左右的边界。又设置了两个变量boolright, boolbottom, 记录当前扫描的方向，扫描方向向右则boolright就是true，扫描方向向下则boolbottom就是true， 由于是顺时针，所以先横方向，再竖直方向，横方向扫描完之后就将行上下的边界改变。然后竖方向扫描完之后就将左右边界改变。注意上下边界改变后下面循环就已经可以直接用现成的边界，不用再加减，最终成功，这是第一次，所以用时较长。
